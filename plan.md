
3-3 구조(총 180석)의 항공기를 기준으로, 승객들이 한 줄로 탑승시도를 할 때 걸리는 시간을 벤치마크해주는 애니메이션을 html, css, js로 구현해야함 

1. 프로젝트 개요
목표: 좌석 배치(30행 × 6열)와 승객의 탑승 전략에 따른 총 소요 시간을 시각화하고 비교함.
전략은 기본적으로 랜덤으로 줄세운 후 탑승했을 때가 디폴트임

핵심 지표: 
- 모든 승객이 자리에 앉을 때까지 걸린 총 시간 (Ticks).
- 짐을 싣거나 먼저 앉은 사람 때문에 줄이 진행되지 않고 대기중인 상황의 빈도 (대기시간, Ticks).

2. 기술 스택 및 구조

HTML CSS로 비행기 내부(복도, 좌석) 레이아웃 구조 정의 및 컨트롤 패널(재생/중지/속도 조절).

좌석 상태(빈 좌석, 탑승 중, 완료) 시각화 및 승객 애니메이션 처리.

틱(Tick) 단위 시뮬레이션 루프. 틱마다 grid를 하나씩 이동하게 하면 좋음

Logic: 승객 객체 생성, 탑승 전략 알고리즘, 충돌 감지 로직.

3. 상세 기능 명세

A. 환경 설정 (Environment)

좌석 배열: Array(30).fill().map(() => Array(6).fill(0)) 형태의 2차원 배열.

복도(Aisle): 승객이 이동하는 경로 (1개 열).

B. 좌석 상태 정의

Empty: 빈 좌석

Occupied: 승객이 앉아 있는 상태

Moving: 이동 중인 승객


C. 승객 객체 (Passenger Object) 속성

targetSeat: 승객이 앉아야 하는 (행, 열) 좌표, 승객마다 좌표 1개 부여

baggageTime: 짐을 선반에 넣는 데 걸리는 값 (기본 1초)

isSeated: 착석 여부. 착석 완료되면 틱이 지나도 움직일 필요가 없음

interference: 창가 자리에 앉아야 하는데 통로 승객이 이미 앉아 있어 비켜줘야 하는 지연 시간 계산.


D. 탑승 전략(Strategy) 선택 기능

사용자가 다음 중 하나를 선택하여 시뮬레이션을 돌릴 수 있어야함

Random (무작위): 랜덤으로 승객을 줄세워서 하나씩 진행시킴

Back to front (뒤에서 앞으로): 뒤쪽 행에 앉는 승객부터 줄세워서 순서대로 입장.

Outside-in (Wilma): 창가(Window) → 가운데(Middle) → 통로(Aisle) 순서.

No assigned seats: 모든 승객은 그냥 좌석과 상관없이 랜덤하게 아무 곳이나 앉음 

Steffen method: 창가 뒤쪽부터 한 칸씩 띄워서 앉히는 이론상 가장 빠른 방식.

block method: 좌석들을 3개의 zone으로 쪼개서 뒷 구역에 속한 승객을 랜덤으로 먼저 입장시킴



4. 시뮬레이션 로직 흐름 (Algorithm)

초기화: 선택한 전략에 따라 승객 리스트의 생성 순서를 섞음(Queue 생성).

이동 단계: 복도에 있는 승객은 앞이 비어 있으면 한 칸 전진.

정지 조건: 자신의 목적지 행(Row)에 도착하면 짐싣기 대기, interference대기 또는 착석을 해야함 

짐 싣기 대기: 승객이 목적지 행에 도착했는데 baggageTime이 있다면 복도를 해당시간동안 점유하며 대기 (이때 뒤 승객들은 모두 멈춤).

interference대기: 승객이 목적지 행에 도착했는데 창가나 가운데 좌석으로 이동해야하는데 중간에 사람이 있다면 지연 발생 

착석: 짐 정리가 끝나면 좌석으로 이동. 

완료: 모든 승객의 isSeated가 true가 되면 타이머 중지.


5. UI/UX 구성 요소

시각화 영역: Canvas 또는 Grid 레이아웃으로 구현된 비행기 평면도.

대시보드: 

- 현재 탑승 완료 인원 / 전체 인원.
- 총 경과 시간 타이머 (Ticks)
- 복도 총 병목 대기시간 (Ticks) 표시.

baggageTime은 모든 승객에 필수로 넣을지, 랜덤으로 특정 승객에만 넣을지 변수로 설정이 가능함

컨트롤러: 탑승 전략 선택 드롭다운, 시작/초기화 버튼, 시뮬레이션 배속 슬라이더.
